## ğŸ–¨ï¸ Thermal Printer Setup

BillPro supports **58mm and 80mm thermal printers** out of the box!

### Supported Printers
- POS-58 Series
- Epson TM Series
- Any ESC/POS compatible printer

### Configuration

Edit `config/printer.json`:

```json
{
    "printer_type": "windows",
    "printer_name": "POS58 Printer",
    "paper_width": 58,
    "cut_paper": true
}
```

> ğŸ’¡ **Tip**: Run `Get-Printer` in PowerShell to find your exact printer name
